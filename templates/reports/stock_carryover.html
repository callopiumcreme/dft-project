Methodology, References & Signature

Methodology. The closing-stock total is computed as the arithmetic sum of K-column aggregate rows that did not receive a production allocation (no entries in the L, M, or O production columns) during the reporting period. Material is physically segregated within the {{ plant_name }} feedstock bay; no commingling with the subsequent period's intake occurs until the stock has been drawn down. This is consistent with the ISCC EU mass-balance method for chain-of-custody accounting (System Document 203, §4) and with the {{ regulator }} expectation that retained intermediates are reported explicitly rather than implied via closure drift.

This Annex D accompanies submission {{ submission_ref }} and is to be read with Annex A — Mass Balance Report ({{ period_label }}) and the cover letter of the same submission bundle. The closing-stock quantity declared on the previous page reconciles, on a kilogram basis, with the difference between the K-aggregate input and the L/M/O production allocations of Annex A for the same period.

For audit purposes, each K-only row in the composition table is traceable to its source position in the production workbook {{ source_workbook }} (column K, row numbers as listed). The supplier-reference column reproduces the ISCC EU chain-of-custody identifier as recorded in the operator's daily log; where a row spans multiple supplier deliveries, the aggregated supplier block reference is shown.
